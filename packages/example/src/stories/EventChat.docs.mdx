# `useEventChat`: 组件通信 `Hooks`

## 介绍

- 提供名字和回调方法，`Hooks` 会返回一个 `emit`
- 用 `emit` 可以向指定名称发送消息，也可以给事件名提供 `callback` 用于接收消息
- 无论是否是父子层级都能相互通信，不会引发不必要的 `rerender`
